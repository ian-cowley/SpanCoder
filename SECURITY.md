# Security Policy

## Supported Versions

We actively provide security patches for the following versions:

| Version | Supported |
| :--- | :--- |
| 1.0.x | :white_check_mark: Yes |
| < 1.0 | :x: No |

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue. Instead, report it privately to ensure we can coordinate a fix before disclosure:

1. Send an email to **spuri@spuri.net**.
2. Include a detailed description of the vulnerability, a proof of concept, and the impact of the exploit.
3. We will acknowledge receipt of your report within 48 hours and work with you to patch the issue promptly.
